## sum-alias
This module exports a simple maths function

  
**Example**
```js
var sum = require("sum-alias")
```


### sum(a, b) ⇒ number ⏏
Sums two numbers together

**Kind**: Exported function  

| Param | Type   | Description   |
| ----- | ------ | ------------- |
| a     | number | first number  |
| b     | number | second number |


**Example**
```js
var result = sum(1, 1)
```
